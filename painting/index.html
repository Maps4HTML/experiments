This example illustrates a few features of the Web-Map-Custom-Element mapml-viewer, as well as some concepts from MapML that it supports:

  • Making whole features into map links: <geometry><map-a href="...">...</map-a></geometry>
  • Linking to a location within the current map using a URL fragment like href="#zoom,longitude,latitude"
  • Different representations of the same feature at different zoom levels / scales.
  • Declarative scaling of features and tiles beyond their 'native' zoom level, to support extremely responsive images (<meta name="zoom" content="min=0,max=7">)
  • Default link style for map-a element (hyperlink blue)
  • Support for the tms link attribute: <link rel="tile" tms tref="...">
